# **App Name**: EstateFind

## Core Features:

- Property Listing Display: Display property listings in a grid layout with property image, title, location, and price on each card.
- Real-time Search: Enable searching properties by location.
- Price Filtering: Allow filtering properties by minimum and maximum price, updating results in real-time.
- Property Details Page: Show the selected property on a dedicated details page.
- API Data Fetch: Fetch properties from the backend API and render them.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to evoke trust and stability in real estate.
- Background color: Very light desaturated blue (#E8EAF6) for a clean, professional look.
- Accent color: Muted violet (#9FA8DA) to complement the primary and create visual interest.
- Body and headline font: 'Inter' sans-serif for a clean and modern design. This font is suitable for both headlines and body text.
- Utilize a grid layout for the property listing page to showcase properties effectively.
- Use subtle transition animations on search results to give an elegant, app-like feel.
